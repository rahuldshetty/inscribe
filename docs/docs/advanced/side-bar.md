# Side Bar Navigation

The sidebar is automatically generated from your documentation files.

## Sorting

Items in the sidebar are sorted by:
1. `weight`: Defined in the front matter (lower numbers first).
2. `title`: Alphabetical sorting if weights are equal.

## Folder Titles

To specify a custom title or weight for a folder in the sidebar, create an `index.md` (or `_folder.yaml` if supported by your version) within that folder.
