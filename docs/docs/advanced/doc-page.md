# Documentation Pages

Documentation pages are organized hierarchically based on your folder structure in the `docs/` directory.

## Structure

Each documentation file should contain front matter to define its title and ordering:

```yaml
---
title: Advanced Usage
weight: 10
---
```

## Hierarchy

The directory structure in `docs/` is reflected in the sidebar. You can use subdirectories to group related pages.
